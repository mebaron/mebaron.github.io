## 七层OSI模型

七层OSI模型有应用层、表示层、会话层、传输层、网络层、数据链路层、物理层；各个层次上的典型网络协议如下：

|    层次    |                 协议                  |
| :--------: | :-----------------------------------: |
|   应用层   |        HTTP、SMTP、FTP、Telnet        |
|   表示层   |        ASCII、MPEG、JPEG、MIDI        |
|   会话层   |       NetBIOS、SAP、SDP、NWLink       |
|   传输层   |             TCP、UDP、SPX             |
|   网络层   |                IP、IPX                |
| 数据链路层 | Ethernet、Token Ring、FDDI、AppleTalk |
